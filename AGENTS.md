# AGENTS.md

## Información del Proyecto
- **Nombre:** DocQR - Sistema de Gestión de Documentos con Código QR
- **Cliente:** Geofal
- **Frontend:** Angular 17+ (Vite/TailwindCSS)
- **Backend:** Laravel 10+ (PHP 8.1+)
- **Base de Datos:** MySQL/MariaDB

## Estructura de Directorios
- `docqr-frontend/`: Aplicación frontend en Angular.
- `docqr-api/`: API REST en Laravel.

## Convenciones de Desarrollo
- Seguir principios SOLID, DRY y KISS.
- Documentar todos los métodos y funciones.
- Mantener cobertura de tests mínima del 80%.
- Commits descriptivos siguiendo el formato Conventional Commits.

## Comandos Útiles
### Desarrollo Local
- **Backend:** `cd docqr-api && php artisan serve`
- **Frontend:** `cd docqr-frontend && npm start`

### Producción
- Ver `DESPLIEGUE_GEOFAL.md` en el directorio raíz.
